# mitter-server

## Prerequisties
- [Go](https://go.dev/doc/install)
- [Air](https://github.com/air-verse/air)

The `go` and `air` commands must be present on PATH in order to run the server and then install the necessary dependencies with:

```bash
go mod download 
```

## Development
Then to begin the local server, run: 

```bash
air
```
